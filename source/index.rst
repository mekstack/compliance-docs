=========================
Welcome to mekstack docs!
=========================


**Mekstack** это большое hyperconverged облако для преподавателей МИЭМ. Поддержка работает 24/7, время ожидания от 1 минуты до 1 дня.
Сервера мекстака располагаются в серверной МИЭМа, поэтому каждому серверу
автоматически предоставляется защита из отдела Информационной Безопасности НИУ ВШЭ.

**Начало работы с облачной платформой мекстак:** :doc:`quick-start`

**Начало работы с сайтом для запуска виртуалок:** :doc:`heat-quick-start`

----
Особенности:
----

* **Cloud Native High Availability Eventual Consistency Monolithic Nocode Nanoservices**

  Использование самых современных технологий : пейсмейкеры, кипалайведы, актив-актив фейловеры, раст, бгп.

* **Software Defined Networking**

  Можно создавать Geneve сети, виртуальные распределенные роутеры, фаерволы и
  лоадбалансеры.

* **SNEEDaaS**

  SNI-Enabled Endpoint Delivery as a Service.
  Сервис для публикации ваших HTTP(S) сервисов в интернете.
  :ref:`sneedaas`

* **VPNaaS**

  Собственный сервис впн для доступа к виртуальным машинам.
  :ref:`vpnaas`

* **Мониторинг**

  Можно смотреть за состоянием облака и всех проектов. `status.mekstack.ru
  <https://status.mekstack.ru/d/YZCsB1Qmz/mekstack-home>`_


* **Собственные образы**

  Можно загружать свои образы. :doc:`images`

-----------
Собираемся внедрить:
-----------

* **Live Migrations**

  Автоматическая балансировка инстансов между серверами для произодительности и стабильность.

* **Интеграция с кабинетом МИЭМ**

  Проекты и их состав синхронизируются с кабинетом автоматически.

* **Зеркала**

  В LAN есть зеркала разных репозиториев.
